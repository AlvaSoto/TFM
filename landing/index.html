Cómo funciona

Conecta el contador

Un emisor de pulsos o tu telelectura existente (LoRaWAN, NB-IoT, exportación CSV de tu AMI). Las lecturas llegan a nuestra API de forma segura.

Aprendemos tu normalidad

Durante 3–4 semanas el sistema construye la línea base de tu instalación: patrón horario, fines de semana, ocupación, temporada. Nada de umbrales genéricos.

El modelo se reentrena con tus datos en menos de un minuto

Alertas que puedes creerte

Doble verificación: la física del contador (el caudal nocturno que nunca para) confirma; la IA amplía la búsqueda. Cada alerta llega con los litros y euros estimados de pérdida.
